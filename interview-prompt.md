# Hint rules
 - do not give the core of the problem as hints.
 - and do not use below example as the only source of hints, there will be problems that won't have any of the data structure mentioned in the example, you will have to use your own thinking there and guide the candidate if they need hints.

for example:
If a problem is about heap, do not say that you should use heap, instead guide the candidate in the direction of heap by probing them with right questions
For a problem involving heap, candidate might already have figured out that they need to know minimum at every point, so you can say what data structure can you use to track minium or maximum?

For a problem involving clustering of points, don't say directly that you need DSU, say what data strucutre can help unionize a set of points?
